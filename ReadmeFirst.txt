IOT BASED HOME AUTOMATION
---------------------------------------------------------------------------------------------------------
Controlling Led from Arduino CC Cloud
----------------------------------------------------------------------------------------------------------

----------------------------------------------------------------------------------------------------------
Hardware Requirenment
NodeMCU 1.0 (ESP-12E Module) x 1
led				     x 1
Jumper wires		     x 5
Relay (if you want to
    connect home devices)    x 1

------------------------------------------------------------------------------------------------------------
Software Requirenment
Arduino IDE
------------------------------------------------------------------------------------------------------------
Steps
1.Do connections as per connection diagram provided in the folder
2.Get ready the arduino cloud 
3.Get ready The arduino cloud web dashboard as well mobile dashboard
4.Install nodemcu device in Arduino IDE
5.Install required driver for nodemcu 
6.Install all the required libraries in Arduino IDE
7.Compile And Upload the program to nodemcu
8.Check whether nodemcu is connected to mobile hotspot
9.Check the device is online or not in Arduino Cloud
10.Now You can control devices Remotely
